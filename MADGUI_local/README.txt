This is a local version of MADGUI, you can use it if the online version is too slow or if you don't want to upload your data on the online server of streamlit.

If you have question or bug report you can use the GitHub or the online MADGUI contact page to contct us.

First you have to pip install all the packages requiered to run the application. You can find them in the requierements.txt file

To launch MADGUI you have to open your terminal then go to the repertory where you save this file:
For exemple if you save the file on your Desktop the line you have to write (on Mac) is :

cd Desktop/MADGUI_local/MADGUI    on MAC
cd Desktop\MADGUI_local\MADGUI    on Windows

Then you have to launch MADGUI by writing :

streamlit run MADGUI.py
